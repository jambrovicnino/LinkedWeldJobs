# LinkedWeldJobs

A job board platform for welding professionals across Europe. Find welding jobs, manage applications, and build your welder profile.

## Tech Stack

- **Frontend:** React 18, TypeScript, Vite, TailwindCSS, ShadCN UI, Zustand
- **Backend:** Express.js, JWT Authentication, JSON File Database
- **Tooling:** React Hook Form + Zod, Axios, Lucide React, Recharts

## Getting Started

### Prerequisites

- Node.js 18+

### Install & Run

```bash
npm install
npm run dev
```

- Frontend: http://localhost:5174
- Backend: http://localhost:3002

## Features

- Browse welding jobs across 28 EU countries
- Apply to jobs and track application status
- Build a comprehensive welder profile with certifications
- Save and bookmark jobs for later
- Job alerts and notifications
- Free subscription plan
